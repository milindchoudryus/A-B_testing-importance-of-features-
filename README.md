# A/B Testing Project

This project demonstrates how to conduct A/B testing using Python.  
It includes hypothesis testing, statistical analysis, and visualizations to determine the impact of changes between control and test groups.

## Files

- `A_B_Testing.ipynb`: Jupyter notebook implementing A/B testing analysis.
- `requirements.txt`: Required Python packages.

## Getting Started

1. Download or clone the repository.
2. Install dependencies with `pip install -r requirements.txt`.
3. Open the notebook and run the analysis.
